# **App Name**: UniHub: Your Bridge to Success

## Core Features:

- Landing Page: A public facing home page providing information about UniHub and guiding users to sign up or log in.
- Student Hireability Score: Calculate a hireability score for students based on their skills, projects, and experience.  Prompt the user for their details if their score hasn't been computed, otherwise display their current score.
- Personalized Learning Path: Generate a personalized learning path for students based on their hireability score using a tool which can analyze their current abilities.
- Internship Recommendations: Recommend internships tailored to the student's profile.
- Course Detail Page: Provide structured, multi-level learning paths for specific skills.
- AI Tutor: A chat interface where students can ask questions on any topic, with document uploading capability, different learning modes and saved chat history.
- Internship Search: A tool allowing students to search internship opportunities with keywords and location filters.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to convey trust and innovation.
- Background color: Light gray (#F0F4F8), for a clean and professional look.
- Accent color: Yellow (#FFD700) to highlight key elements.
- Headline font: 'Space Grotesk' (sans-serif) for a techy and modern look.
- Body font: 'Inter' (sans-serif) to complement 'Space Grotesk' (sans-serif), ensuring readability and consistency.
- Use consistent and modern icons related to education, career, and AI, ensuring a professional look.
- Clean and organized layouts with clear sections for each feature.
